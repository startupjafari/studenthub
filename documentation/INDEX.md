## Основные концепции

### Архитектура

StudentHub использует модульную архитектуру на базе NestJS. Каждый модуль отвечает за определенную область функциональности.

### Аутентификация

Система использует JWT токены (access и refresh) для аутентификации. Поддерживается email верификация и двухфакторная аутентификация.

### База данных

PostgreSQL используется как основная база данных. Prisma ORM обеспечивает типобезопасный доступ к данным.

### Real-time

WebSocket (Socket.io) используется для real-time функций: сообщения, уведомления, групповые чаты.
